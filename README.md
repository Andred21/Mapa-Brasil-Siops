# Mapa-Brasil-Siops
Sistema web interativo para análise de dados públicos de saúde, integrado à API SIOPS. Permite visualizar indicadores estaduais e municipais, séries históricas, mapas interativos e painéis dinâmicos com gráficos e cálculos automáticos. Desenvolvido em Laravel, React, Inertia e Tailwind.
